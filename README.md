## Grammar
> 📂How to add a file or a folder📂
```
  ADD(檔案名稱,檔案類型);
```
> Delete a file?
```
  DELETE(檔案名稱);
```
> Then...put your file in the folder?
```
  PUT(檔案名稱,資料夾名稱)
```
